# ISC-S1
Glosario

1.Actuadores: Son dispositivos que convierten una señal eléctrica en movimiento, como los motores en robots o máquinas.

2.APIS: Son herramientas que permiten que diferentes programas o aplicaciones se comuniquen entre sí y compartan información.

3.Backend: Es la parte de un sistema o aplicación que funciona en segundo plano y se encarga de procesar datos y controlar el funcionamiento interno.

4.Backups: Son copias de seguridad de la información que se guardan para evitar perder los datos importantes.

5.Ciberseguridad: Es la protección de computadoras, sistemas y datos contra ataques, robos o accesos no autorizados a través de internet.

6.Embebido: Es un sistema que está integrado dentro de un dispositivo para cumplir una función específica, como el sistema que controla un microondas o un carro.

7.Escalabilidad: Es la capacidad de un sistema para crecer y adaptarse a más usuarios o más trabajo sin dejar de funcionar correctamente.

8.Firmware: Es un programa que está instalado dentro de un dispositivo electrónico y controla su funcionamiento básico, como en una impresora o una computadora.

9.Hitos: Son eventos importantes o momentos clave que marcan un avance o cambio significativo en el desarrollo de algo, como la tecnología.

10.loT:  Es la conexión de objetos a internet, como relojes inteligentes, cámaras o electrodomésticos, para que puedan enviar y recibir información.

11.Perifericos: Son dispositivos que se conectan a una computadora, como el teclado, el mouse, la impresora o el monitor.

12.Phishing: Es un tipo de fraude en el que una persona intenta robar información personal, como contraseñas o datos bancarios, haciéndose pasar por una entidad confiable.

13.UEFI: Es un sistema que ayuda a iniciar la computadora y permite que el sistema operativo funcione correctamente desde el encendido.

14.Valvulas: Son componentes electrónicos antiguos que se utilizaban en las primeras computadoras para controlar la electricidad antes de que existieran los transistores.

15. BIOS: Es un programa que está integrado en la computadora y se encarga de iniciar el sistema cuando se enciende.
